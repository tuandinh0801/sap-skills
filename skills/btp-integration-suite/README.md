# SAP BTP Integration Suite Skill

Comprehensive skill for developing and operating enterprise integration solutions using SAP Integration Suite on SAP Business Technology Platform.

## Capabilities Covered

- **Cloud Integration** (CPI/SCPI) - Integration flow development
- **API Management** (APIM) - API proxy and policy management
- **Event Mesh** - Event-driven architecture
- **Edge Integration Cell** - Hybrid/on-premise deployments
- **Integration Advisor** - B2B mapping automation
- **Trading Partner Management** - B2B partner onboarding
- **Graph** - Unified business data API
- **OData Provisioning** - OData service creation
- **Integration Assessment** - ISA-M methodology
- **Migration Assessment** - SAP PO migration planning
